# Analisis-Harga-Motor-Bekas

Proyek mini ini bertujuan untuk menganalisis dan mengidentifikasi faktor-faktor kunci yang memengaruhi harga jual kembali (resale value) motor bekas. Dengan menggunakan data dari dealer motor, analisis ini berfokus pada eksplorasi hubungan antara karakteristik teknis motor (seperti $\text{usia}$, $\text{jarak tempuh}$, dan $\text{kapasitas mesin}$) dengan harga jualnya di pasar. Hasil analisis ini memberikan wawasan pasar yang terperinci bagi dealer, pembeli, maupun penjual untuk menentukan harga yang kompetitif dan strategis.
